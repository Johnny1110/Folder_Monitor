# Log_Monitor_Note

<br>

---

<br>

這個套件的作用是監控一個指定目錄中的檔案異動狀況，並把這些資料存入到 database 中。同時，當檔案內的資料有新增行數(例如 log 多印出一行)也會馬上輸出新的行數資料。
